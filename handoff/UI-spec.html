面向 H5 / Android 原生开发人员 | 版本 1.0 | 2026-06-26

线上预览：https://flyingfish-accelerator.vercel.app

1. 项目概述

飞鱼加速器是一款面向用户的网络加速工具，核心功能：一键加速、模式/线路选择、积分体系、免费会员兑换。

  • 设计尺寸：375 x 812 (iPhone X 基准，适配 Android)
  • 主题：深色模式 (Deep Ocean)，无浅色模式
  • 字体：-apple-system, BlinkMacSystemFont, PingFang SC, Helvetica Neue, Microsoft YaHei, sans-serif
  • 圆角基准：12px (卡片/按钮)
  • 状态栏：预留 48px 高度

4. 首页 (HomePage)

顶部栏

  • 标题："飞鱼加速器" 渐变文字，16px bold
  • 右上角："送会员"按钮
  • 按钮样式：圆角药丸，border primary/20，内边距 px-3 py-1.5
  • 按钮内容：Share2图标(14px) + "送会员"文字(12px)

状态徽章

  • 位置：标题下方居中
  • 样式：圆角药丸，px-4 py-2
  • 默认：primary/8背景 + "准备就绪"
  • 连接中：primary/10背景 + "连接中..."
  • 已连接：connected/10背景 + "加速中 HH:MM:SS"

火箭动画区

  • 尺寸：280 x 245px
  • SVG矢量图，包含：仪表盘弧线 + 刻度 + 火箭
  • 连接时：火箭上移7px + 火焰动画
  • 仪表盘：160°弧线，21个刻度线
  • 火焰：3层椭圆叠加 + 闪烁动画(0.35s)

加速按钮

  • 宽度：85%（最大360px）
  • 高度：55px
  • 圆角：16px (rounded-2xl)
  • 文字：18px，不加粗(font-normal)
  • 图标：Zap(默认) / Square(已连接)，20px
  • 文案：立即提速 / 连接中... / 停止加速
  • 按下效果：scale(0.96)

模式/线路卡片

  • 布局：2列网格(grid-cols-2)，间距12px
  • 背景：glass-card
  • 内边距：14px
  • 图标：32x32圆角方块 + 4px图标
  • 标签：10px muted色
  • 值：12px medium 前景色
  • 右侧：ChevronRight箭头

广告位

  • 最小高度：80px
  • 虚线边框：1px dashed muted/20
  • 内边距：16px
  • 占位文字："广告位预留"

12. 技术备注

  • PWA支持：manifest.json 已配置，支持添加到主屏幕
  • 响应式：以375px为基准，按钮使用 max-width 约束
  • 状态栏：顶部预留48px，适配刘海屏/挖孔屏
  • 安全区域：底部导航预留24px paddingBottom，适配全面屏
  • 滚动：自定义滚动条（4px宽，深色轨道）
  • 图标库：Lucide React → Android 可使用 SVG 转 VectorDrawable
